Trabalho Final de Disciplina:
Sistema de Estoque e Vendas para Pequenos Comércios

Requisitos Funcionais:
Cadastrar novos produtos. |
Listar todos os produtos cadastrados. |
Pesquisar um produto pelo nome ou código. |
Alterar informações de um produto.|
Registrar vendas, reduzindo a quantidade em estoque. |
Excluir produtos cadastrados. |
Encerrar a execução pelo menu principal. |
Requisitos Não Funcionais:
Os dados serão armazenados em memória durante a execução do programa. |
O sistema deverá possuir um menu simples e de fácil utilização. |
